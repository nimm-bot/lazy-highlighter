Lazy highlighter is initialized in script as so:
document.querySelectorAll('[data-lazy-highlighter-activator]').forEach(function (elem) {
    new LazyHighlighter(elem, {});
});

Class constructor takes 2 arguments, the second of which is a JSON object, containing widget options.

Current version of lazy highlighter accepts these option properties:

{

  • highlightedTemplate (string) - HTML to wrap highlighted element. If provided, MUST contain a {highlighted} substring as substitute for highlighted element. Passing empty string or '{highlighted}' removes the wrapper.
    Defaults to:
    <div data-lazy-highlighter-container><div data-lazy-highlighter-background></div>{highlighted}</div>
  • events (JSON object) - Events to bind when initializing a widget for activator element. Keys must match DOM events, such as mouseover or click. Values are functions correcponding to said events.
    Defaults to highlighted elements highlighted class toggle on mouseenter and mouseleave.
  • preset (string) - if given, assigns one of preset classes to highlighted element's container, but only if the container has data-lazy-highlighter-container attribute. Used for quick styling. Available values are 'MARKER', 'BORDER' and 'OVERLAY'.

}
